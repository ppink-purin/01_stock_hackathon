3. 맛보고 피드백하면 수정됨

> 😊 "말로 설명하면 만들어준다고?"

# 🆚 ChatGPT/Claude 웹 vs Claude Code

| 구분 | ChatGPT / Claude 웹 | **Claude Code** |
|:---|:---|:---|
| 환경 | 웹 브라우저 채팅창 | **내 컴퓨터 터미널** |
| 할 수 있는 것 | 코드 "알려주기" | 코드 **직접 쓰기** |
| 파일 접근 | 불가 | **내 프로젝트 폴더 직접 수정** |
| 실행 | 복사-붙여넣기 필요 | **바로 실행 + 결과 확인** |
| 비유 | 📖 요리책 읽어주는 친구 | 🍳 **직접 요리해주는 셰프** |

> ChatGPT가 "이렇게 하면 돼요"라면,
> Claude Code는 **"제가 해놨어요, 확인해 보세요"** 🎯

# 💬 실제 프롬프트 ① — 프로젝트의 시작

> 맨 처음 입력한 프롬프트 (2/11 14:06)

```
- 주제: "주식도사" 에이전트 만들기
- 개발요건:
 * 모바일 웹사이트에서 사용자의 질문에 대해 대답
   예) "키움증권 지금 사면 좋을까?", "삼성전자 거래 현황"
 * 질문에 대한 답변 정보는 반드시 '다음 금융' 사이트에 있는
   정보만 활용할 것
 * 에이전트는 다음과 같은 단계로 사고하고, 각 단계별 사고 과정
   및 결과를 사용자에게 알려줄 것
   1) 질문의 의도 분석
   2) 탐색 계획 수립
   3) web_fetch 형태로 정보 스크랩하여 분석 및 요약
   4) 초보 투자자가 이해하기 적합한 최종 답변 생성
 * vercel을 이용하여 웹서비스 배포
```

> 🎯 **목표 + 기능 + 데이터 소스 + 사고 단계 + 배포 방식**까지 한 번에 전달

# 💬 실제 프롬프트 ⑤ — 조사 먼저, 결정은 내가

```

활용할 수 있는 방법이 없을까?
가장 단순무식한 웹스크래이핑 방식을 포함해서
한번더 다음 금융 사이트 정보를 활용할 수 있는
최선의 방법을 찾아봐줘.
```


> 🎯 "조사는 AI가, **결정은 내가**" — 가장 효율적인 협업 패턴

# 💬 실제 프롬프트 ⑥ — AI한테 분석을 맡기기

```
자체적으로 로컬 테스트를 하면서
```

```
답변 제공시 '다음 금융에서 제공이 불가한 관계로
대체 API(네이버 금융)를 사용하였습니다'는 내용이 명시되게 해줘.
```

> 🎯 분석도 AI가, 하지만 **UX 결정(안내 문구)은 내가** 지시

> 강의 교안 6개를 한 번에 생성 (2/12 23:36)

```
지금까지 구현된 내용으로
코딩, AI, 에이전트, tool-calling, user-analytics 를 주제로
이 프로젝트에서 개발된 내용을 바탕으로
각 주제별로 1시간 분량으로 개념 설명과
실제 코드로 개발된 내용 예시를 디테일하게 설명하는
강의 슬라이드를 만들어줘.
각 주제별로 강의 계획을 세우고,
강의 컨텐츠를 제작하는 전문가 역할을 할
sub-agents를 정의하여 생성해주고,
각 주제별로 각 sub-agent 별로
parallel하게 작업을 시켜줘.
전체 강의 교안이 완성되기 전까지는
멈추거나 나의 의견을 묻지 말고 계속 진행시켜줘.
```

> 🎯 **5개 주제 x 병렬 에이전트** → 한 번에 강의 교안 5개 동시 생성!

# 패턴 정리 — 실제 사용한 7가지 방법

| # | 패턴 | 실제 예시 (요약) |

# ❌ vs ✅ — 실제 프롬프트로 비교

<div class="columns">
<div>

- "챗봇 만들어줘"
- "예쁘게 해줘"
- "에러 고쳐줘"
- "분석 기능 넣어줘"

</div>
<div>

### ✅ 내가 실제로 입력한 것들
- "주식도사 에이전트를 만들건데, 다음 금융 사이트 데이터만 활용하고, 4단계 사고 과정을 보여줘"
- "피넛츠 코믹스 만화책에서 캐릭터간 대화하는 듯한 스타일로 바꿔줘"
- "오류가 발생했습니다: Claude Code executable not found... 분석 및 조치해줘"
- "KPI를 제안해주고, 이벤트 로그를 저장 관리하는 기능을 추가해줘"

</div>
</div>

> 🎯 핵심: **무엇을 + 왜 + 어떤 조건으로** 말해주면 됩니다

```
    1단계                2단계              3단계

 검토하게 해줘"       네이버는 유지하고"
```
